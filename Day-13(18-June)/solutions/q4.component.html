Q4 – markAllAsTouched() Demo

What does markAllAsTouched() do?

Angular only reveals validation errors for fields the user has interacted with ("touched"). Calling markAllAsTouched() inside onSubmit() forces every field into the touched state, making all error messages visible at once when the user tries to submit an incomplete form.

Username is required. Username must be at least {{ f['username'].errors?.['minlength'].requiredLength }} characters.
Password is required. Password must be at least {{ f['password'].errors?.['minlength'].requiredLength }} characters.

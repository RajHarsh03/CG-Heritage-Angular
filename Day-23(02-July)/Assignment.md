## Task: Student Class with Object & Template Literal

Create a `Student` class with three properties — `id`, `name`, and `course`. Instantiate an object of the class and display the student details in an Angular component using a template literal.

---

## Requirements

- Define a `Student` class with:
  - `id` — number
  - `name` — string
  - `course` — string
  - A `constructor` that sets all three properties
- Create a `student` object using `new Student(...)`
- Build a template literal string in the format:
  `Student ID: 1 | Name: Harsh | Course: Angular`
- Display both the individual properties and the template literal output in the Angular template

---

## Expected Output

| Field | Value |
|---|---|
| ID | 1 |
| Name | Harsh |
| Course | Angular |

**Template Literal:** `Student ID: 1 | Name: Harsh | Course: Angular`
